How to make a token (1 min)
GitHub → Settings → Developer settings → Fine-grained tokens → Generate new token → Repository access: Only select repositories → your JCWLUNACY repo → Permissions → Repository → Contents: Read and write → Generate. Paste it above. It is stored only in this browser (localStorage) and sent only to api.github.com.
